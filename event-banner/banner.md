# Sewercide CTF Challenge

Welcome to the **Sewercide Plumbing** network.

Your task is to find and retrieve the hidden flag from the target server.

- **Attacker machine:** Kali Linux 2025.2
- **Target:** A web server running somehwere on the network
- **Objective:** Gain access, escalate privileges, and capture the flag file
- **Flag format:** `flag{something}`

Good luck and happy hacking!
